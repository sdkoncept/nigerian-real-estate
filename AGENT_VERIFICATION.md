# Agent Verification Requirements

## Overview

To become a verified agent on the Nigerian Real Estate Platform, agents must submit specific documentation proving their credentials and identity. This verification process ensures platform safety and builds trust with property seekers.

## Required Documents

Agents must submit the following documents:

### 1. Government-Issued ID (REQUIRED)
- **What**: Valid government-issued identification document
- **Accepted Forms**:
  - National ID Card
  - International Passport
  - Driver's License
  - Voter's Card
- **Format**: PDF, JPG, or PNG
- **Requirements**:
  - ID must be current and valid
  - Full name must match the name on the account
  - Photo must be clearly visible
  - ID number must be readable

### 2. Business Registration (OPTIONAL but Recommended)
- **What**: CAC Certificate of Incorporation or Business Name Registration
- **Format**: PDF, JPG, or PNG
- **Requirements**:
  - Must be current and valid
  - Company/business name must match profile
- **Note**: This helps agents achieve "Trusted Agent" status

### 3. Professional Credentials (OPTIONAL)
- **What**: Additional professional certifications or credentials
- **Examples**:
  - Real Estate Professional Certifications
  - Property Management Certifications
  - Training Certificates
- **Format**: PDF, JPG, or PNG

## Document Submission Process

### Step 1: Complete Agent Profile
Before submitting documents, agents must:
1. Verify their email address
2. Verify their phone number
3. Complete their agent profile with:
   - Full name
   - Company name (if applicable)
   - Bio/description
   - Specialties
   - Years of experience
   - Contact information
   - Location (city, state)

### Step 2: Upload Documents
Agents can upload verification documents through:
- **Route**: `/agent/verification` (Agent Dashboard)
- **Process**:
  1. Select document type (License, ID, Credentials)
  2. Upload file (max 5MB per file)
  3. Add optional notes
  4. Submit for review

### Step 3: Admin Review
- Documents are reviewed by platform administrators
- Review typically takes 2-5 business days
- Agents receive email notifications about status updates

### Step 4: Verification Status
After review, agents receive one of three statuses:

#### ✅ Verified (Tiered System)
Agents can achieve different verification tiers:

**Platform Verified** (Level 1):
- ID verified
- Phone & email verified
- Basic verification badge
- Can list properties

**Trusted Agent** (Level 2):
- All Level 1 requirements
- Bank account verified (optional)
- Business registration (optional)
- "Trusted Agent" badge
- Higher search ranking

**Premium Agent** (Level 3):
- All Level 2 requirements
- 5+ successful transactions
- 4.0+ rating
- "Premium Agent" badge
- Priority placement

**Top Agent** (Level 4):
- All Level 3 requirements
- 20+ properties sold
- 4.5+ rating
- "Top Agent" badge
- Featured placement

#### ⏳ Pending
- Documents submitted but not yet reviewed
- Agent can still use platform with limited features
- Status updates via email

#### ❌ Rejected
- Documents did not meet requirements
- Agent receives rejection reason
- Can resubmit with corrected documents

## Document Requirements Checklist

### Government ID (REQUIRED)
- [ ] ID is current and not expired
- [ ] Full name matches account name
- [ ] Photo is clearly visible
- [ ] ID number is readable
- [ ] File is clear and readable (not blurry)

### Professional Credentials (if applicable)
- [ ] Certificate is current
- [ ] Issuing organization is recognized
- [ ] Name matches account name
- [ ] File is clear and readable

## File Upload Guidelines

### Accepted Formats
- **Images**: JPG, JPEG, PNG
- **Documents**: PDF
- **Maximum File Size**: 5MB per file
- **Maximum Total Size**: 15MB (all documents combined)

### File Naming
Use descriptive names:
- `license_REA_LAG_2020_001.pdf`
- `national_id_john_doe.jpg`
- `certificate_property_management.pdf`

### Image Quality
- Documents must be clear and readable
- Avoid blurry or low-resolution images
- Ensure good lighting when photographing documents
- Scan documents when possible (better quality than photos)

## Common Rejection Reasons

Documents may be rejected if:
1. **ID Issues**:
   - ID is expired
   - Name doesn't match account name
   - Photo is not clearly visible
   - ID is illegible or blurry

2. **File Issues**:
   - File is corrupted or unreadable
   - File size exceeds limit
   - Wrong file format
   - File is not actually a document (wrong file uploaded)

3. **Other Issues**:
   - Documents appear to be fake or altered
   - Information doesn't match profile details
   - Missing required documents

## After Verification

Once verified, agents can:
- ✅ Display verification badge on profile (tiered system)
- ✅ List properties (unlimited for Premium/Top agents)
- ✅ Access analytics dashboard
- ✅ Receive priority in search results (higher tiers get more priority)
- ✅ Build trust with potential clients
- ✅ Get 3x more inquiries than unverified agents
- ✅ Access premium features (based on tier)

## Resubmission Process

If verification is rejected:
1. Review rejection reason (sent via email)
2. Correct any issues
3. Upload new/corrected documents
4. Resubmit for review
5. Wait for admin review (2-5 business days)

## Privacy & Security

- All documents are stored securely
- Documents are only accessible to:
  - The agent (their own documents)
  - Platform administrators (for review)
- Documents are encrypted in storage
- Documents are never shared with third parties
- Documents can be deleted upon account closure

## Legal Compliance

By submitting verification documents, agents agree:
- Documents are authentic and belong to them
- They have the right to use these documents
- Information provided is accurate
- They will notify platform of any changes to their license status
- Platform may verify documents with issuing authorities
